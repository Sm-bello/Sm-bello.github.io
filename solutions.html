SYS · 01 In Development

Lifestone

Sermon Intelligence Desktop

A Tauri + Rust desktop app that transcribes sermons live and fully offline, using whisper-rs for speech-to-text and Aho-Corasick + ONNX embeddings to detect scripture references in real time against a local database of 311,074 verses across 10 translations. Detected verses fire simultaneously to NDI broadcast, a PowerPoint sidecar, and a dedicated second-monitor verse screen.

SYS · 04 Operational

Penelope AI Agent

Sovereign, offline-first AI agent

A Python agent that doesn't depend on any single provider — cascading across Kimi, DeepSeek, Groq, Cerebras, GLM-4-Plus, SambaNova, and more, with automatic fallback to a fully local Ollama stack when every cloud path fails. A ReAct reasoning loop drives tool use, task classification, and domain-specific validation, and a hardware fingerprint locks the agent to its host machine.
