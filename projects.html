Landing Gear Digital Twin Architecture

A physics-based simulation aimed at predicting oleo-pneumatic strut failure. By modeling the internal gas and hydraulic dynamics in Simulink, this project seeks to create a "Virtual Twin" that compares real-time sensor data against ideal performance curves to detect seal degradation before leaks occur.

Flight Telemetry Stack

A full data pipeline connecting FlightGear (Simulation) to a Grafana dashboard. Uses Python to parse UDP packets and InfluxDB to store high-frequency time-series data (pitch, roll, altitude) for post-flight analysis.

Structural Beam Analysis
